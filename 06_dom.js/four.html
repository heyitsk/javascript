//++++++APPEND CHILD++++++++ this appends the child in the specified parent element for ex here body is the parent element and we are appending divelement in the body. You can also asign an element as soon as its created for ex:- document.body.appendChild(document.createElement('p')). You can also use nested append child const fragment = document.createDocumentFragment(); const li = fragment .appendChild(document.createElement("section")) .appendChild(document.createElement("ul")) .appendChild(document.createElement("li")); li.textContent = "hello world"; document.body.appendChild(fragment); this will form a section element and in it will be nested ul and in it will be nested li with textcontent as hello world if an element is already created and you want to append in it you can selectit using querySelector and then use .appendchild()
